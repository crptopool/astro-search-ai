    translation="നീയും നിന്‍റെ കൂടെയുള്ളവരില്‍ ഒരു വിഭാഗവും രാത്രിയുടെ മിക്കവാറും മൂന്നില്‍ രണ്ടു ഭാഗവും (ചിലപ്പോള്‍) പകുതിയും (ചിലപ്പോള്‍) മൂന്നിലൊന്നും നിന്നു നമസ്കരിക്കുന്നുണ്ട്‌ എന്ന്‌ തീര്‍ച്ചയായും നിന്‍റെ രക്ഷിതാവിന്നറിയാം. അല്ലാഹുവാണ്‌ രാത്രിയെയും പകലിനെയും കണക്കാക്കുന്നത്‌. നിങ്ങള്‍ക്ക്‌ അത്‌ ക്ലിപ്തപ്പെടുത്താനാവുകയില്ലെന്ന്‌ അവന്നറിയാം. അതിനാല്‍ അവന്‍ നിങ്ങള്‍ക്ക്‌ ഇളവ്‌ ചെയ്തിരിക്കുന്നു. ആകയാല്‍ നിങ്ങള്‍ ഖുര്‍ആനില്‍ നിന്ന്‌ സൌകര്യപ്പെട്ടത്‌ ഓതിക്കൊണ്ട്‌ നമസ്കരിക്കുക. നിങ്ങളുടെ കൂട്ടത്തില്‍ രോഗികളും ഭൂമിയില്‍ സഞ്ചരിച്ച്‌ അല്ലാഹുവിന്‍റെ അനുഗ്രഹം തേടിക്കൊണ്ടിരിക്കുന്ന വേറെ ചിലരും അല്ലാഹുവിന്‍റെ മാര്‍ഗത്തില്‍ യുദ്ധം ചെയ്യുന്ന മറ്റ്‌ ചിലരും ഉണ്ടാകും എന്ന്‌ അല്ലാഹുവിന്നറിയാം. അതിനാല്‍ അതില്‍ (ഖുര്‍ആനില്‍) നിന്ന്‌ സൌകര്യപ്പെട്ടത്‌ നിങ്ങള്‍ പാരായണം ചെയ്തു കൊള്ളുകയും നമസ്കാരം മുറപ്രകാരം നിര്‍വഹിക്കുകയും സകാത്ത്‌ നല്‍കുകയും അല്ലാഹുവിന്ന്‌ ഉത്തമമായ കടം നല്‍കുകയും ചെയ്യുക. സ്വദേഹങ്ങള്‍ക്ക്‌ വേണ്ടി നിങ്ങള്‍ എന്തൊരു നന്‍മ മുന്‍കൂട്ടി ചെയ്ത്‌ വെക്കുകയാണെങ്കിലും അല്ലാഹുവിങ്കല്‍ അത്‌ ഗുണകരവും ഏറ്റവും മഹത്തായ പ്രതിഫലമുള്ളതുമായി നിങ്ങള്‍ കണ്ടെത്തുന്നതാണ്‌. നിങ്ങള്‍ അല്ലാഹുവോട്‌ പാപമോചനം തേടുകയും ചെയ്യുക. തീര്‍ച്ചയായും അല്ലാഹു ഏറെ പൊറുക്കുന്നവനും കരുണാനിധിയുമാകുന്നു."
