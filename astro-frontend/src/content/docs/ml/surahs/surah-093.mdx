<Aside type="reference" title="2">

<Aside type="quran" title="3">
مَا وَدَّعَكَ رَبُّكَ وَمَا قَلَىٰ
</Aside>

<Aside type="reference" title="3">
( നബിയേ, ) നിന്‍റെ രക്ഷിതാവ്‌ നിന്നെ കൈവിട്ടിട്ടില്ല. വെറുത്തിട്ടുമില്ല.
</Aside>

<Aside type="quran" title="4">
وَلَلْءَاخِرَةُ خَيْرٌۭ لَّكَ مِنَ ٱلْأُولَىٰ
</Aside>

<Aside type="reference" title="4">
തീര്‍ച്ചയായും പരലോകമാണ്‌ നിനക്ക്‌ ഇഹലോകത്തെക്കാള്‍ ഉത്തമമായിട്ടുള്ളത്‌.
</Aside>

---

<Aside type="quran" title="5">
وَلَسَوْفَ يُعْطِيكَ رَبُّكَ فَتَرْضَىٰٓ
</Aside>

<Aside type="reference" title="5">
വഴിയെ നിനക്ക്‌ നിന്‍റെ രക്ഷിതാവ്‌ ( അനുഗ്രഹങ്ങള്‍ ) നല്‍കുന്നതും അപ്പോള്‍ നീ തൃപ്തിപ്പെടുന്നതുമാണ.്‌
</Aside>

---

<Aside type="quran" title="6">
أَلَمْ يَجِدْكَ يَتِيمًۭا فَـَٔاوَىٰ
</Aside>

<Aside type="reference" title="6">
നിന്നെ അവന്‍ ഒരു അനാഥയായി കണെ്ടത്തുകയും , എന്നിട്ട്‌ ( നിനക്ക്‌ ) ആശ്രയം നല്‍കുകയും ചെയ്തില്ലേ?
</Aside>

---

<Aside type="quran" title="7">
وَوَجَدَكَ ضَآلًّۭا فَهَدَىٰ
</Aside>

<Aside type="reference" title="7">
നിന്നെ അവന്‍ വഴി അറിയാത്തവനായി കണെ്ടത്തുകയും എന്നിട്ട്‌ ( നിനക്ക്‌ ) മാര്‍ഗദര്‍ശനം നല്‍കുകയും ചെയ്തിരിക്കുന്നു.
</Aside>

---

<Aside type="quran" title="8">
وَوَجَدَكَ عَآئِلًۭا فَأَغْنَىٰ
</Aside>

<Aside type="reference" title="8">
നിന്നെ അവന്‍ ദരിദ്രനായി കണെ്ടത്തുകയും എന്നിട്ട്‌ അവന്‍ ഐശ്വര്യം നല്‍കുകയും ചെയ്തിരിക്കുന്നു.
</Aside>
