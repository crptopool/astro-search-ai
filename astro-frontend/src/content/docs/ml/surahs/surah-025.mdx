<Aside type="reference" title="1">
തന്‍റെ ദാസന്‍റെ മേല്‍ സത്യാസത്യവിവേചനത്തിനുള്ള പ്രമാണം (ഖുര്‍ആന്‍) അവതരിപ്പിച്ചവന്‍ അനുഗ്രഹപൂര്‍ണ്ണനാകുന്നു. അദ്ദേഹം (റസൂല്‍) ലോകര്‍ക്ക്‌ ഒരു താക്കീതുകാരന്‍ ആയിരിക്കുന്നതിനു വേണ്ടിയത്രെ അത്‌.
</Aside>

<Aside type="reference" title="2">
ആകാശങ്ങളുടെയും ഭൂമിയുടെയും ആധിപത്യം ആര്‍ക്കാണോ അവനത്രെ ( അത്‌ അവതരിപ്പിച്ചവന്‍. ) അവന്‍ സന്താനത്തെ സ്വീകരിച്ചിട്ടില്ല. ആധിപത്യത്തില്‍ അവന്ന്‌ യാതൊരു പങ്കാളിയും ഉണ്ടായിട്ടുമില്ല. ഓരോ വസ്തുവെയും അവന്‍ സൃഷ്ടിക്കുകയും, അതിനെ അവന്‍ ശരിയാംവണ്ണം വ്യവസ്ഥപ്പെടുത്തുകയും ചെയ്തിരിക്കുന്നു.
</Aside>

<Aside type="reference" title="3">

</Aside>
